# school-grade-management
# 學生成績管理系統（Student Management System）

這是一個使用 HTML + CSS + JavaScript 製作的網頁版學生成績管理系統。

---

## 📌 功能介紹
- 新增學生（姓名 + 成績）
- 刪除學生資料
- 搜尋學生（依姓名）
- 顯示所有學生列表
- 計算平均成績
- 使用 localStorage 儲存資料（重新整理不會消失）

---

## 🛠 使用技術
- HTML
- CSS
- JavaScript
- localStorage
